# 🛡️ 융보공 보안 포트폴리오 템플릿

성신여자대학교 융합보안공학과 학술동아리 융보공 학생들을 위한 포트폴리오 웹사이트 템플릿입니다.

## 🚀 빠른 시작

### 1. 프로젝트 설치
```bash
npm install
npm run dev
```

### 2. 브라우저에서 확인
http://localhost:3000

## 📝 커스터마이징

### 개인 정보 수정
`src/app/data/portfolioData.js` 파일에서 모든 개인 정보를 수정할 수 있습니다.

### 이미지 추가
모든 이미지 파일은 `public` 폴더에 넣어주세요.

## 📚 상세 가이드
자세한 커스터마이징 방법은 [가이드 문서](./GUIDE.md)를 참고하세요.

## 🚀 배포
Vercel을 통해 무료로 배포할 수 있습니다.

## 🛠️ 기술 스택
- Next.js 15
- React 19
- Tailwind CSS 4
- Lucide React
